Altiverr Webhook Relay

API Endpoints

  • /api/slack-webhook/[webhook-id] - Relay for Slack webhooks to n8n
  • /api/webhook - General webhook handler
  • /api/queue - Queue management endpoint
  • /api/delete-message - Message deletion endpoint

Slack Webhook Relay

This service relays webhooks from Slack to n8n by:

  1. Handling Slack's URL verification challenge
  2. Forwarding all webhook events to the configured n8n instance
  3. Preserving all headers and request data

To use, set your Slack webhook URL to:

https://altiverr-webhook-relay.vercel.app/api/slack-webhook/[your-webhook-id]
